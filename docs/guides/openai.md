# Guide: OpenAI Credentials

This guide explains how to obtain the required credentials for the OpenAI integration.

## Required Secrets

-   `api_key`

## Instructions

### 1. API Key

The API key is the standard method for authenticating with the OpenAI API.

-   **Official Documentation**: [OpenAI API Docs: API Keys](https://platform.openai.com/docs/api-reference/authentication)

-   **Steps**:
    1.  Log in to your OpenAI account at [platform.openai.com](https://platform.openai.com/).
    2.  Navigate to the **API Keys** section in the left-hand menu.
    3.  Click **Create new secret key**.
    4.  Give the key a name (optional).
    5.  Click **Create secret key** and copy the key value immediately. It will not be shown again.
