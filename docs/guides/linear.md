# Guide: Linear Credentials

This guide explains how to obtain the required credentials for the Linear API.

## Required Secrets

-   `api_key`

## Instructions

### 1. API Key

Linear uses personal API keys for authentication.

-   **Official Documentation**: [Linear Docs: API](https://developers.linear.app/docs/api)

-   **Steps**:
    1.  Log in to your Linear account.
    2.  Navigate to **Settings** > **API**.
    3.  Under the **Personal API Keys** section, click **Create key**.
    4.  Give the key a label (e.g., "Vault Integration").
    5.  Click **Create**.
    6.  Copy the API key value immediately. It will not be shown again.
