# Guide: Anthropic Credentials

This guide explains how to obtain the required credentials for the Anthropic (Claude) API.

## Required Secrets

-   `api_key`

## Instructions

### 1. API Key

-   **Official Documentation**: [Anthropic Developer Docs: Get Access](https://docs.anthropic.com/claude/reference/getting-started-with-the-api)

-   **Steps**:
    1.  Log in to your Anthropic account dashboard.
    2.  Navigate to the **API Keys** section in your account settings.
    3.  Click **Create Key**.
    4.  Give the key a name.
    5.  Click **Create Key** and copy the value immediately. It will not be shown again.
