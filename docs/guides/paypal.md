# Guide: PayPal Credentials

This guide explains how to obtain the required credentials for the PayPal REST API.

## Required Secrets

-   `client_id`
-   `client_secret`

## Instructions

### 1. Create a REST API App

Credentials are generated by creating an app in the PayPal Developer Dashboard.

-   **Official Documentation**: [PayPal Developer Docs: Get started](https://developer.paypal.com/api/rest/get-started/)

-   **Steps**:
    1.  Log in to the [PayPal Developer Dashboard](https://developer.paypal.com/dashboard/).
    2.  Navigate to the **My Apps & Credentials** section.
    3.  Ensure you are in the **Live** tab (not Sandbox) for production credentials.
    4.  Click **Create App**.
    5.  Give the app a name and click **Create App**.
    6.  The next screen will show you the `client_id` and a `client_secret` for your new app. Copy these values.
